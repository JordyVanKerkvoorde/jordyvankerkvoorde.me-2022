n my second year of applied computer science at HoGent we had to create our own webapplication and REST API using Angular and a .NET API. The purpose of my application was to check the availability of parking spots in parking garages. The data for this was supplied by an API from Stad Gent. However I wanted to store this data to show it on a chart and in a later stage make predictions on them. To do this I made use of an extra technology. I wrote a NodeJS server application to fetch the data and store it to a MSSQL database. This database was then accessed by the web application using a .NET API. At the front-end of the application I used OpenLayers to visualise the parking garages in the system on a map. To show the data on a chart I used ChartJS. The application also has login and register functionality.
