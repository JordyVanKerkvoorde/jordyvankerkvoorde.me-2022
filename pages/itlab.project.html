In my second year of applied computer science at HoGent we had to make a web and desktop application for the ITLab. We had to make a web application using ASP.NET Core and a desktop application using Java both using a shared MS SQL database. For the web app we had to make a calendar and reservation system. The idea was that students could attend events from the ITLab when registering for it online. Notifications about the events would also be posted on the web app. The admin has access to a different page that allows him to get an overview of the registered users and the ability to mark students that subscribed as if they where present for the session or not.

The desktop application was used to schedule new events, to mail a notification or event to the students, to manage roles and permissions for other users and to show analytics.
